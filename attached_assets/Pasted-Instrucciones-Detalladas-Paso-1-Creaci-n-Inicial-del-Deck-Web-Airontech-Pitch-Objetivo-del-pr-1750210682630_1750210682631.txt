Instrucciones Detalladas Paso 1 – Creación Inicial del Deck-Web (Airontech Pitch)
🔹 Objetivo del proyecto
Desarrollar un deck interactivo y visualmente impactante para presentar Airontech a inversionistas. Debe ser profesional, claro y elegante.

🔹 Stack tecnológico requerido
Usa estrictamente estas tecnologías:

Next.js con React y TypeScript (por rendimiento, SEO y facilidad de escalabilidad).

Tailwind CSS (para estilos rápidos, limpios y profesionales).

Framer Motion (para animaciones suaves y modernas).

Google Fonts: preferentemente Montserrat o Inter.

ESLint configurado para mantener código ordenado.

🔹 Detalles visuales (IMPORTANTE)
Usa Tailwind para un diseño limpio, con fuentes modernas y tamaño grande para fácil lectura.

Tipografía recomendada: Montserrat o Inter (Google Fonts).

Paleta sugerida (elegante, profesional y tecnológica):

Fondo: gris oscuro (#111827 o similar)

Textos principales: blanco (#FFFFFF)

Acentos: azul eléctrico (#3B82F6) o verde corporativo (#10B981)

Asegúrate que el layout sea 100% responsive (escritorio, tablet y móvil).

🔹 Interactividad básica
El slide debe ocupar toda la pantalla del usuario y ajustarse automáticamente.

Asegura que el scroll esté bloqueado inicialmente y que la interacción sea únicamente mediante botones o teclas (flechas arriba y abajo).

🔹 Criterios de éxito
Al terminar este paso, verifica claramente:

Proyecto Next.js funcional.

Tailwind correctamente configurado.

Framer Motion animando entrada de textos y botón.

SlideIntro perfectamente visible y centrado.

Responsividad validada en escritorio y móvil.

Código limpio y organizado según estructura.
